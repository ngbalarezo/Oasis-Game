# Oasis: Text-Based Adventure RPG

Oasis is a text-based adventure RPG created for a college data structures class. As it stands the game is fully functional and modular in structure so that it can be fleshed out with creative content. Creative content includes fully customizable wilderness and village locations for the map; customizeable Church, shop, and lounge buildings; NPCs and dialogues; weapons, armor, potions, and misc. items. The number of custom objects is limited only by your PC's processing power.
